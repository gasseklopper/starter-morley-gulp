Flexbox grid#

Layout includes flexbox based responsive grid system with 12 row.

col-12 (100%)
col-9 (75%)
col-6 (50%)
col-3 (25%)
<!-- flexbox grid example -->
<div class="container">
  <div class="row">
    <div class="column col-6">col-6</div>
    <div class="column col-3">col-3</div>
    <div class="column col-2">col-2</div>
    <div class="column col-1">col-1</div>
  </div>
</div>

Add the row class to a container with the container class. And add any element you want with the column class inside the container. These row will take up the space equally. You can specific the width of each column by adding class col-<1-12>.

Gapless grid#

And you can add the col-gapless class to the row to have gapless row.

col-6 (gapless)
col-6 (gapless)

  <div class="column col-6">col-6</div>
</div>

By default, Spectre grid has multi-line flexbox enabled. You can add the col-oneline class to row to make all its child row positioned in the same single row.

col-8
col-8
<!-- one line row -->

  <div class="column col-8">col-8</div>
